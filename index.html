ALL THE CHATBOTS

unofficial api to all the web chatbots i could find!

open issues to suggest new bots or if you want your bot removed

install

pip install all_the_chatbots

usage

there is a method called ask_X , where X is the bot name for every bot

from all_the_chatbots import *

bot_says = ask_mitsuku("hello")
print(bot_says)

you can also get data about available bots

from all_the_chatbots import bot_list, bot_number, bot_map

print(bot_number())
print(bot_list())
print(bot_map())

Notable Bots

It's easy to make a bot, but you will notice most of them use the same base aiml files and are plain dumb

The following chatbots are worth of notice

NeuralConvo - Deep learning chatbot trained through Neuralconvo, a Torch library that implements Sequence to Sequence Learning with Neural Networks (seq2seq), reproducing the results in the Neural Conversational Model paper (aka the Google chatbot).

Mitsuku - Mitsuku, a four-time winner of the Loebner Prize Turing Test, is the world's best conversational chatbot.

Wolfram Alpha - Compute expert-level answers using Wolfram's breakthrough algorithms, knowledgebase and AI technology

Aristo - An intelligent system that reads, learns, and reasons about science.

Euclid - solve SAT-style math questions

AskTheCatterpillar - Ask The Caterpillar is a harm reduction chatbot that allows people easy access to information about substances so that they can make informed choices.
